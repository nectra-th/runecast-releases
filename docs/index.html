Getting Started

Up and Running in 60 Seconds

1. Download & Launch ▶

Download RuneCast.exe and run it. Windows SmartScreen may show a warning — click "More info" then "Run anyway". The app requests admin privileges for window capture access.

2. Select a Window ▶

Click "Select Window" in the toolbar. A list of all open windows appears. Pick the one you want to automate — it can be a game, browser, or any application.

3. Draw a Template ▶

Choose a mode (Click, Drag, Wait, HotKey) from the sidebar, then draw two rectangles on the live preview: the search area and the target image. RuneCast saves them as a template.

4. Adjust Threshold ▶

Each template has a match bar showing real-time confidence. Drag the threshold slider to control how strictly RuneCast matches. Higher = more precise, lower = more tolerant.

5. Hit Play ▶

Press F9 or click Play. RuneCast continuously scans and performs actions when templates match. Press F9 again to stop. Use "BG" mode to automate without moving your real mouse.

FAQ

Frequently Asked Questions

Is RuneCast safe to use? ▶

Yes. RuneCast uses pixel-level image recognition only — it never reads or injects into game memory. It's a single .exe with no installer, no registry modifications, and no background services.

Will I get banned for using this in games? ▶

RuneCast operates at the pixel level and does not modify game files or memory. However, automating games may violate their Terms of Service. Use responsibly and check each game's policy.

What does "lifetime updates" mean? ▶

Every future update is included at no additional cost. The app automatically checks for updates on startup. You always get the latest features and improvements — forever.

What apps and games does it work with? ▶

RuneCast works with any application that renders to a window on Windows 10/11 — games, browsers, desktop apps, and more.

Can I use my PC while it runs? ▶

Absolutely. Background mode sends clicks and keystrokes directly to the target window without moving your real mouse. You can browse, chat, or work on other tasks simultaneously.

Can I share templates with friends? ▶

Yes! Use the Export/Import feature in the gear menu. Templates are bundled into a .runecast file with all images included. Screen resolution is checked on import to ensure compatibility.
